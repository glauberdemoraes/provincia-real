⚙️  CONFIGURAÇÃO AVANÇADA

   Mudar intervalo de polling:
   → src/pages/Dashboard/index.tsx
   → pollingIntervalMs: 15000 (15 segundos)
   
   Mudar intervalo do cron:
   → sync-service.mjs
   → cron.schedule('*/15 * * * *') para 15 minutos
   
   PM2 para produção:
   $ npm install -g pm2
   $ pm2 start sync-service.mjs --name "sync-service"
   $ pm2 startup
   $ pm2 save

🎯 IMPORTANTE: SSH ATIVO SEM SOBRECARREGAR

   ✅ Frontend polling: 30 segundos (leve)
   ✅ Cron service: heartbeat a cada 1 minuto (muito leve)
   ✅ Realtime listeners: passivos (sem tráfego se sem mudanças)
   
   SSH NÃO cai porque:
   • Polling mantém conexão ativa
   • Heartbeat cron mantém ativo
   • Sem sobrecarregar servidor
