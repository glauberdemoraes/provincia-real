$ npm install -g pm2
$ pm2 start sync-service.mjs --name "sync-service"

📞 SUPORTE RÁPIDO
═══════════════════════════════════════════════════════════════════════════

Problema: Mount sync não rodando
Solução: Verificar se enableMountSync: true em Dashboard.tsx

Problema: Dados não mudam ao clicar refresh
Solução: Abrir Console (F12), ver se há erro de rede

Problema: Cron service caiu
Solução: Ver tail -50 .sync-service.log, reiniciar com npm run sync:service

Problema: SSH drop
Solução: Usar Cron service (npm run sync:service), que mantém ativo

Problema: Muitos logs no console
Solução: É normal, mudar if (verbose) em performFullSync se quiser desabilitar
